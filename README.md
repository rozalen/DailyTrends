# DailyTrends
Daily Trends is a small application that show principal news from different newspapers

## How run the project?

``` bash
# run containers
docker-compose up

# run lumen migrations
docker exec dailytrends_api_1 php artisan migrate

Open browser on http://localhost:8080

```
